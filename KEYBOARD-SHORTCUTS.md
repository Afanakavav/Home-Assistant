# ⌨️ Tastiera Rapida - Peronciolillo Home Assistant

Questa guida elenca tutte le scorciatoie da tastiera disponibili nell'app.

## Scorciatoie Globali

Queste scorciatoie funzionano in tutte le pagine (quando non si sta digitando in un campo di input):

### Navigazione
- **`Ctrl + 1`** - Vai alla Dashboard
- **`Ctrl + 2`** - Vai alle Spese
- **`Ctrl + 3`** - Vai alla Lista Spesa
- **`Ctrl + 4`** - Vai ai Task
- **`Ctrl + 5`** - Vai all'Inventario

### Azioni Rapide
- **`Ctrl + N`** - Aggiungi nuova spesa
  - Se sei nella pagina Spese, apre il dialog
  - Altrimenti, naviga alla pagina Spese
- **`Ctrl + T`** - Aggiungi nuovo task
  - Se sei nella pagina Task, apre il dialog
  - Altrimenti, naviga alla pagina Task
- **`Ctrl + S`** - Aggiungi alla lista spesa
  - Se sei nella pagina Lista Spesa, apre il dialog
  - Altrimenti, naviga alla pagina Lista Spesa

## Note

- Le scorciatoie **non funzionano** quando stai digitando in un campo di input, textarea o elemento editabile
- Su **macOS**, usa **`Cmd`** invece di **`Ctrl`**
- Le scorciatoie sono visibili anche nei tooltip dei pulsanti FAB (Floating Action Button)

## Suggerimenti

- Usa le scorciatoie per navigare rapidamente tra le sezioni
- Le scorciatoie per aggiungere elementi sono particolarmente utili per inserimenti rapidi
- I tooltip sui pulsanti mostrano sempre la scorciatoia disponibile
